# PeekPanel

Chrome拡張機能のサイドパネルで複数のWebページを閲覧・管理できるブラウザツールです。

---

## 概要

PeekPanelは、メインブラウザの横にサイドパネルを開き、その中で複数のタブを管理しながらWebページを閲覧できる拡張機能です。AIサービス（Gemini、Claude、ChatGPT、Grok）へのクイックアクセスや、テキスト選択からの即座なAI問い合わせなど、効率的なブラウジングをサポートします。

---

## 主な機能

### サイドパネルブラウザ

- **複数タブ管理**: サイドパネル内で複数のWebページをタブとして開き、切り替えて閲覧できます
- **タブグループ**: 関連するタブをグループにまとめて整理できます
- **履歴管理**: 閉じたタブの履歴を保存し、必要に応じて復元できます
- **タブの自動スリープ**: 一定時間アクティブでないタブを自動的にスリープ状態にし、メモリ使用量を削減します
- **ブックマーク**: よく使うページをブックマークし、フォルダで整理できます

### ナビゲーション

- **戻る/進む**: 各タブ内でのページ遷移履歴をナビゲートできます
- **URLバー**: URLを直接入力してページにアクセスできます
- **Google検索**: URLバーにキーワードを入力するとGoogle検索を実行します
- **リロード**: 現在表示中のページを再読み込みします
- **Switch to Main Browser**: サイドパネルで表示中のページをメインブラウザの新しいタブで開きます

### AIサービス連携

- **デフォルトAIタブ**: 初回起動時にGemini、Claude、ChatGPT、Grokのタブが自動的に作成されます
- **AI切り替え**: ドロップダウンから使用するAIサービスを選択できます
- **テキスト選択からAI問い合わせ**: Webページ上のテキストを選択し、右クリックメニューからAIに送信できます

### カスタムプロンプト

テキスト選択時の右クリックメニューに表示されるプロンプトをカスタマイズできます。

**デフォルトプロンプト:**
- 清書する
- 要約する
- 英語に翻訳
- ビジネス文書化

**カスタムプロンプト:**
- 独自のプロンプトを追加できます
- 各プロンプトの有効/無効を切り替えられます
- プロンプト内で `{text}` を使用すると、選択したテキストに置き換わります

### コンテキストメニュー

**ページ上での右クリック:**
- 「Open PeekPanel」: 現在のページをサイドパネルに移動します

**テキスト選択時の右クリック:**
- 「Googleで「{選択テキスト}」を検索」: 選択テキストでGoogle検索を実行します
- カスタムプロンプトの一覧が表示され、選択したAIサービスに送信できます

## 設定

設定ページでは以下の項目を変更できます。

### 基本設定

| 項目 | 説明 |
|------|------|
| 自動送信 | AIにテキストを送信後、自動的に送信ボタンを押します |
| 使用するAI | プロンプト送信時に使用するAIサービスを選択します |

### タブスリープ設定

| 項目 | 説明 |
|------|------|
| 自動スリープ | 一定時間操作のないタブを自動的にスリープ状態にします |
| スリープまでの時間 | 自動スリープが発動するまでの時間（分）を設定します |
| スリープ禁止ドメイン | 自動スリープを適用しないドメインを設定します（YouTube、Twitchなど） |

### テーマ設定

| 項目 | 説明 |
|------|------|
| システム設定に従う | OSのダークモード設定に連動します |
| ライトモード | 常にライトテーマを使用します |
| ダークモード | 常にダークテーマを使用します |

## 操作方法

### サイドパネルを開く

1. ブラウザのツールバーにあるPeekPanelアイコンをクリックします
2. サイドパネルが開き、デフォルトのAIタブが表示されます

### 新しいタブを開く

1. タブバー右端の「+」ボタンをクリックします
2. URLバーにURLまたは検索キーワードを入力してEnterキーを押します

### タブの操作

| 操作 | 方法 |
|------|------|
| タブ切り替え | タブをクリック |
| タブを閉じる | タブを中クリック、または右クリックメニューから「タブを閉じる」 |
| タブの複製 | 右クリックメニューから「タブを複製」 |
| タブをピン留め | 右クリックメニューから「タブをピン留め」 |
| タブのミュート | 右クリックメニューから「タブをミュート」 |

### タブグループの操作

| 操作 | 方法 |
|------|------|
| 新規グループ作成 | タブの右クリックメニューから「新しいグループに追加」 |
| グループに追加 | タブの右クリックメニューから既存のグループを選択 |
| グループの展開/折りたたみ | グループヘッダーをクリック |
| グループ名の変更 | グループヘッダーを右クリックして「グループ名を変更」 |

### テキスト選択からAIに送信

1. Webページ上でテキストを選択します
2. 右クリックしてコンテキストメニューを開きます
3. 使用したいプロンプトを選択します
4. 選択したAIサービスが開き、テキストが自動入力されます

## 技術仕様

### アーキテクチャ

- **Manifest V3**: Chrome拡張機能の最新仕様に準拠
- **モジュール構成**: ES Modules によるクリーンなコード分離
- **SVGアイコン**: 統一されたインラインSVGアイコンセット（絵文字不使用）
- **テーマ対応**: CSS変数によるダーク/ライトモード完全対応

## 注意事項

- 一部のWebサイトはセキュリティ上の理由により、iframe内での表示がブロックされる場合があります
- サイドパネル内での全画面表示には制限があります
- `chrome://`で始まるブラウザ内部ページは表示できません
- 他の拡張機能（アドブロッカー等）はサイドパネル内のページには適用されません（Chrome拡張の仕様制限）
